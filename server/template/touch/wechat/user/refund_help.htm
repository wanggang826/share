退款帮助

1.提现成功后待退款是什么？

待退款指您已经提现成功，金额转到待退款里，正在进行退款中，但退款金额还未到账，提现的金额到账后，待退款将清零，微信提示退款到账。

2.提现的金额退回哪里？

提现成功后，押金将原路径退回，使用微信零钱支付，则退回微信零钱。银行卡支付，则退回支付银行卡。

3.退回到银行卡的金额会有银行短信提醒吗？

金额原路径返回，银行不会有短信提醒，您可在收到微信到账通知时查看银行流水进行确认。

4.为何押金退回需要2－5个工作日？

退款首先返回给支付平台，微信退款需要1-2个工作日；2）支付平台返还给银行，银行退款需要1－3个工作日；3）银行把钱打到个人账户，需要约3－5个工作日。

5.微信提示退款到账后，在哪里查询退款？

微信零钱支付，可在微信“钱包”→“零钱”→“零钱明细”/“钱包”→“交易记录”里进行查询。
